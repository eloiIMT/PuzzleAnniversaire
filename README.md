# 🎁 Puzzle d'Anniversaire pour Margaux

Coucou Margaux! 

Voici un petit cadeau spécial créé avec amour pour ton anniversaire: **un puzzle interactif en ligne**! 

Complète le puzzle 4x3 en glissant les pièces dans les bonnes positions, et découvre une surprise spéciale à la fin! 🎉

## 🎮 Jouer au puzzle

👉 **[Clique ici pour accéder au puzzle](https://eloiIMT.github.io/PuzzleAnniversaire/)**

## ✨ Comment jouer?

1. Glisse une pièce du puzzle
2. Place-la sur une case vide adjacente
3. Continue jusqu'à ce que tout le puzzle soit complété
4. Découvre tes cadeaux! 🎁
